// How to get your token (2 minutes)
1. Go to github.com/settings/tokens/new — you need a free GitHub account if you don't have one already. 2. In the Note field type something like WatchNexus QA so you remember what it's for. 3. Set Expiration to 90 days (or no expiration if you prefer a one-time setup). 4. Under Select scopes, check public_repo — that is the only permission needed to post issues to this project. 5. Scroll down and click Generate token. Copy it immediately — GitHub only shows it once. 6. Paste it into the Your Personal Access Token field above and click Test Connection. Your token is saved in your browser only — it is never stored in or transmitted with the report file.
⚠ Keep your token private. Do not share it or paste it anywhere other than this field. If you think it has been exposed, go to github.com/settings/tokens and delete it immediately.
